Bug reports and feature requests

As quests

The preferred way to provide feedback, request features, report bugs, or start any Questhub-related conversations is by posting them to Meta realm:

  1. Create a quest there.
  2. Invite me (<%= partial.user({ login: "berekuk" }) %>) to the quest.
You can then Leave the quest for me to pick up, or you can stay on quest's team, and if I implement it, we both'll get some points.

If you're unsure if the task is already in works, check out the list of existing quests. Don't worry too much about creating duplicates, though!

As Github issues

If your issue is that you can't sign in to Questhub, or the above instruction is for some reason inconvenient for you, you can also create an issue on Github.

Or talk to me directly

If you'd prefer a private conversation, just send me an e-mail: me@berekuk.ru.
